📚 Book Review Service (bookr_review)

Spring Boot와 JPA를 활용한 도서 리뷰 관리 서비스입니다. 단순히 기능을 구현하는 것에 그치지 않고, Front-end와 Back-end 간의 안전한 데이터 통신과 XSS 공격 방어 등 보안적인 측면을 고려하여 개발되었습니다.

🛠 Tech Stack
Backend: Java 17, Spring Boot 3.x, Spring Data JPA

Database: MySQL

Frontend: HTML5, CSS3, JavaScript (Vanilla JS), jQuery

Authentication: Session-based Authentication
